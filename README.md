## Installation

You can install SwiftyBox in two ways:

1. Build from source with Xcode.
2. Download the latest `.ipa` from [Releases](https://github.com/cranci1/SwiftyBox/releases).

Direct download for `v1.0.1`:

- https://github.com/cranci1/SwiftyBox/releases/download/v1.0.1/SwiftyBox.ipa
